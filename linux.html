LE SYSTÈME D'EXPLOITATION LINUX

Linux est un système d'exploitation libre et gratuit. Il sert de base à de nombreuses distributions utilisées aujourd'hui.

Nature et Philosophie

Linux est développé en "Open Source", ce qui veut dire que son code est public et que tout le monde peut participer à son amélioration.

Caractéristiques Techniques

  • Modèle économique libre et collaboratif
  • Bonne sécurité contre les virus
  • Utilise peu de ressources matérielles
  • Très stable pour les serveurs

L'HISTOIRE DE LINUX

De 1991 à aujourd'hui, Linux est passé d'un petit projet à un système utilisé partout dans le monde.

Naissance d'Unix

Ken Thompson et Dennis Ritchie créent Unix dans les années 70. C'est l'ancêtre de Linux.

Projet GNU

Richard Stallman commence le projet GNU pour créer un système d'exploitation libre.

Le message de Linus

Linus Torvalds, étudiant finlandais, annonce qu'il travaille sur un petit système d'exploitation.

Tux est né

Linux choisit un manchot comme mascotte officielle après avoir été mordu par un manchot au zoo.

Debian & Slackware

Les premières distributions importantes apparaissent et structurent l'écosystème Linux.

Ubuntu

Mark Shuttleworth lance Ubuntu, rendant Linux accessible à tout le monde.

Android

Google lance Android, basé sur le noyau Linux. Linux devient le système le plus utilisé au monde sur mobile grâce à Android.

Les Distributions

Linux existe sous différentes "versions" appelées distributions. Chacune a ses particularités.

Ubuntu

La plus populaire. Idéale pour les débutants, stable et facile à utiliser.

Pour : Débutants, Développeurs

Debian

Le père de nombreuses distributions. Réputée pour sa stabilité à toute épreuve.

Pour : Serveurs, Puristes

Arch Linux

Mise à jour continue. Vous construisez votre système vous-même. Pour les experts.

Pour : Experts, Passionnés

Linux Mint

Basée sur Ubuntu, elle ressemble à Windows pour faciliter la transition.

Pour : Ceux qui viennent de Windows

Fedora

À la pointe de l'innovation. Souvent utilisée par Linus Torvalds lui-même.

Pour : Développeurs, Innovateurs

Trisquel

100% libre. Recommandée par la Free Software Foundation. Aucun code propriétaire.

Pour : Partisans du logiciel libre

OÙ UTILISE-T-ON LINUX ?

Linux est présent dans de nombreux domaines de notre vie quotidienne.

Serveurs & Infrastructure

La plupart des sites web utilisent Linux sur leurs serveurs car il est stable et fiable.

Téléphones mobiles

Android est basé sur Linux, ce qui en fait le système le plus utilisé sur les téléphones.

Objets connectés

On trouve Linux dans les routeurs, les box internet, les voitures connectées...

Supercalculateurs

Tous les 500 supercalculateurs les plus puissants du monde utilisent Linux.

Exploration spatiale

La NASA utilise Linux pour piloter des satellites et des robots comme Ingenuity sur Mars.
